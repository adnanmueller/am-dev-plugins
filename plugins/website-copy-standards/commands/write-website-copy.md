# Write Website Copy

You are executing the `/write-website-copy` command.

@../SKILL.md

Guide the user through the full copywriting workflow defined above.

## Process

1. **Identify Audience** — Ask B2B or B2C, gather target persona details
2. **Map Entities** — Define primary entities, attributes, and relationships
3. **Structure** — Create outline with question H2s for voice/AI search
4. **Write Emotion-First** — Headlines with visceral appeal, body with logical support
5. **Apply Answer-First** — Question trigger + direct answer (40-60 words)
6. **Implement Schema** — Generate appropriate JSON-LD

## Arguments

- `$ARGUMENTS` — Optional page type (landing, about, product, service)

## Output

Deliver copy that:
- Converts humans (emotional triggers + logical support)
- Gets cited by AI (entity-mapped, answer-first structure)
- Passes accessibility checks
- Includes JSON-LD Schema markup
