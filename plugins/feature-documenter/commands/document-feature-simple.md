# Document Feature (Simple)

You are executing the `/document-feature-simple` command.

@../SKILL.md

Create feature documentation using the **simplified template**.

## When to Use

- Small, self-contained features
- Quick reference documentation
- Features with minimal dependencies
- When time is limited

## Arguments

- `$ARGUMENTS` — Feature name or file path to document

## Template Reference

Use `references/template-simple.md` as the base structure.

## Sections

1. **Problem Statement** — What problem does this feature solve?
2. **Solution Overview** — How does it solve it?
3. **Core Components** — What are the key parts?
4. **Usage Example** — How do you use it?

## Output

Concise feature documentation focused on essentials, saved to the project's docs folder.
